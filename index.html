How it works

Takes about two minutes to set up. Nothing to install.

Mission Momentum runs in your browser and calls Anthropic's API directly with your own key. Nothing is stored on our servers, and unlike some of our other tools, there's no Google Sheet or script to set up.

1

Get an API key

Go to console.anthropic.com and create a key. New accounts include $5 in trial credits. Follow the full setup guide: Mac or Windows.

2

Paste it in

Paste your key into any diagnostic once. It's stored only in your browser and works across all five tools automatically.

3

Start diagnosing

Paste your copy into any diagnostic and get scored feedback, flagged issues, and rewrites in under a minute.

FAQ

Common questions.

Does Mission Momentum cost anything?

The tools themselves have no subscription, no account, and no data stored on our servers. AI analysis is billed per use directly by Anthropic through your own API key. For occasional use, costs are typically a few cents per diagnostic.

Do I need an API key?

Yes. Every diagnostic runs through Claude, so an Anthropic API key is required to use any of the five tools. There's no free-tier option without one.

Do I need a Google account or spreadsheet?

No. Unlike Network Momentum, Mission Momentum doesn't store anything, so there's no Google Sheet or Apps Script to set up. Just an API key.

Which AI provider does Mission Momentum use?

Mission Momentum is built on Anthropic's Claude. It doesn't currently support other AI providers.

Is my data private?

Yes. Your copy is sent directly from your browser to Anthropic using your own API key when you run a diagnostic. Nothing passes through or is stored on our servers, and nothing is saved after you close the tab.
