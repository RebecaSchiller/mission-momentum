Setup Guide · Mac

Get set up on Mac

Three steps, about five minutes, nothing to install. You'll need an Anthropic API key and a Mac web browser (Chrome, Safari, or Firefox all work).

1

Get an Anthropic API key

Mission Momentum runs entirely in your browser and calls Anthropic's API directly with your own key — there's no Mission Momentum account to create.

  1. Go to console.anthropic.com and sign up, or log in if you already have an account.
  2. Once signed in, go to Settings → API Keys (or the "API Keys" link in the left sidebar).
  3. Click Create Key, give it a name (e.g. "Mission Momentum"), and click Create.
  4. Copy the key that appears. It starts with sk-ant-. This is the only time the full key is shown, so copy it now.
Note: New Anthropic accounts typically include a small amount of free trial credit, but exact amounts and terms can change — check console.anthropic.com for current details. After any trial credit is used, diagnostics are billed per use, typically a few cents each.
2

Save your key in Mission Momentum

You only need to do this once — it works across all five diagnostics automatically.

  1. Go to missionmomentum.app.
  2. Scroll to the "How it works" section and find the box labeled "Your Anthropic API key."
  3. Paste your key (⌘+V) into the field and click Save key.

Your key is stored only in your Mac's browser storage for that browser. It's never sent to Plain Speaking Communications or stored on any server — only to Anthropic, directly from your browser, when you run a diagnostic.

3

Run your first diagnostic

Pick whichever tool matches what you're working on:

  • Donor Appeal — for fundraising emails and appeal letters
  • Thank-You Letter — for donor acknowledgment and stewardship copy
  • Lapsed Donor — for win-back and re-engagement messages
  • Positioning — for mission statements, about pages, and elevator pitches
  • Newsletter — for newsletters and impact reports

Paste your copy into the tool, fill in any optional context, and click the button to run the diagnostic. You'll get a score across six categories, flagged issues tied to your actual wording, and rewrites for your two weakest sections — usually in under a minute.

Troubleshooting

"No API key found" banner won't go away. Go back to missionmomentum.app, re-paste your key into the "Your Anthropic API key" box, and click Save key again. Each browser stores its own copy, so if you switch browsers or use a private/incognito window, you'll need to save the key there too.

Diagnostic returns an error. Double-check your key starts with sk-ant- and that it was copied in full. If it looks right, check your usage and billing status at console.anthropic.com — a diagnostic will fail if trial credit has run out and no billing method is on file.

Key stopped working after a browser update or cleanup. Browser storage can be cleared by cache-clearing tools or privacy settings. If that happens, just repeat Step 2 to save your key again.

Ready to go

Head back and save your key to start diagnosing your copy.
